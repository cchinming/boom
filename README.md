# boom
中国文化景点散点图分布
  其中包含mapp.py  download.py UserAgent.pyc文件
  
  而download.py运行环境python3   ；    mapp.py运行环境python2
    先运行download进行数据的抓取，将数据存.csv文件
    
    在使用mapp进行文件的分析会图
    
    此程序由于只抓取了文化古迹的景点数据，除此之外还可以进行更多数据的抓取
